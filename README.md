　　买到卡后，别急着插卡上网！90%的连接问题，出在APN设置上。这是最关键的一步，却最常被忽略。

　　插卡后，通常会自动收到配置信息（记得允许弹出），点击安装即可。如果没收到，或网络还是不通，就需要手动设置：

-   **Digicel APN**: `digicelwap`
-   **Flow APN**: `internet` 或 `flowwap`

　　（具体参数可能会有微调，以运营商提供的为准。想象一下，这就像给新钥匙配一把锁，对了才能打开数据的大门。）
